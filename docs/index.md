# YouTube Lounge JS Documentation

**YouTube Lounge JS** is a JavaScript library to interact with the unofficial and undocumented YouTube Lounge API.
This allows interaction with YouTube apps across device types such as a smart TV or gaming consoles.
The API has functions for controlling the device and to receive status updates from the device.

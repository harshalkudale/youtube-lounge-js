# API Reference

The library is organized into several modules, all available under the `src/` directory.

## Main Wrapper

The primary entry point for the library is the `YtLoungeApi` class in `src/wrapper.js`.

## API Module

`src/api.js` contains low-level functions for interacting with the YouTube Lounge API endpoints.

## Discovery Module

`src/dial.js` handles device discovery using the DIAL protocol.

## Event Listener Module

`src/event_listener.js` defines the base `EventListener` class for handling screen events.

## Events Module

`src/events.js` defines the various event classes emitted by the library.

## Models Module

`src/models.js` and `src/lounge_models.js` contain data models for authentication, devices, and commands.

## Utilities Module

`src/util.js` contains internal utility functions.

## Exceptions Module

`src/exceptions.js` defines custom error types.
